Question 10 : What is a bootstrap card and how would you create one?

The .card class is used to create a card container.

The .card-img-top class is used to place an image at the top of the card.

The .card-body class is used to contain the main content of the card.

Inside the .card-body, you can add various card content, such as a title (card-title), text (card-text), and a button (btn).
