Question 8 : What is === operator?

It is used to compare two values for equality, both in terms of value and data type.

The === operator returns true if the values being compared are equal in both value and data type, and false otherwise.


Ex:- <p id="example">

<script>
let x = 5;
document.getElementById("example").innerHTML = (x === 5);
<script/>

Ans:-true

Ex:- <p id="example1">

<script>
let y = 5;
document.getElementById("example1").innerHTML = (y === "5");
<script/>

Ans:-false
