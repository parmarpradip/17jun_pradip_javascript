Question-16 : How can a page be forced to load another page in JavaScript?

To force a page to load another page in JavaScript, you can use the window.location object. The window.location object provides information about the current URL of the browser and allows you to navigate to a new URL.


window.location.href:
- You can set the href property of window.location to the URL of the page you want to load. This method is commonly used for redirection.
Ex:-
// Redirect to another page
window.location.href = 'https://www.example.com';
