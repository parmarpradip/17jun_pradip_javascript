Question-14 : What are all the types of Pop up boxes available in JavaScript?

In JavaScript, there are three types of popup boxes commonly used for interacting with the user: alert, confirm, and prompt.

Alert Box (alert):

Displays a message to the user in a dialog box.


Ex:-

alert("This is an alert box!");

Confirm Box (confirm):

Displays a message and provides the user with "OK" and "Cancel" buttons.


Ex:-

let result = confirm("Do you want to proceed?");
if (result) {
// User clicked "OK"
console.log("User clicked OK");
} else {
// User clicked "Cancel"
console.log("User clicked Cancel");
}

Prompt Box (prompt):

Displays a message and provides an input field for the user to enter text.

Ex:-

let userInput = prompt("Please enter your name:", "hello");
if (userInput !== null) {
console.log("User entered: " + userInput);
} else {
console.log("User clicked Cancel");
}
