Question 5 : What are undeclared and undefined variables?

1.Undeclared Variables:
An undeclared variable is a variable that has not been explicitly declared using a declaration statement, such as `var`, `let`, or `const`, before it is used.
ex:
console.log(x); // ReferenceError: x is not defined

2.Undefined Variables:
An undefined variable is a variable that has been declared but has not been assigned a value. It exists in the program's scope, but it doesn't have a defined value yet.
ex:
var y;
console.log(y); // undefined
